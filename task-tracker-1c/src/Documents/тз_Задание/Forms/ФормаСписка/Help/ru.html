Трекер заданий

Общий список заданий, в который выведены следующие колонки:

  • Наименование
  • Номер
  • Состояние
  • Текущий исполнитель
  • Приоритет
  • Дата смены состояния - когда в последний раз у задания изменялось состояние.
  • Автор состояния - кто изменил состояние в последний раз.
  • Дата создания
  • Вид задания
  • Автор
  • Проект
  • Этап

В случае, если в профиле, установленном в отборе списка, определены виды аналитики, включенные для отображения в списке, то в списке заданий будут автоматически сформированы колонки, соответствующие заданным видам аналитик.

Какие быстры отборы предусмотрены в форме?
  • Проект - по проекту задания.
  • Этап - по этапу задания.
  • Состояние – по текущему состоянию задания.
  • Текущий исполнитель – по текущему исполнителю задания.
