Задание

Информация, доступная для указания в задании
  • Наименование - краткое описание задания для идентификации.
  • Содержание – подробное описание задания.
  • Профиль - профиль заданий, определяющий доступные состояния, приоритеты и виды заданий.
  • Проект - проект, в рамках которого осуществляются поставленные задачи.
  • Этап - этап, в рамках которого осуществляются поставленные задачи.
  • Текущий исполнитель - кто непосредственно выполняет задание или работу в данный момент.
  • Состояние - текущая стадия выполнения задания.
  • Основание - задание, на основании которого было сформировано текущее задание.
  • Вид задания - классификация задания.
  • Приоритет - степень важности или срочности задания. 

В случае, если в профиле определены и настроены виды аналитик, они будут автоматически выведены в задании.

Статусы выполнения и состояния задания

Задание может находиться в одном из 3 статусов выполнения:

  • Новая
  • В работе
  • Завершена

Статусы выполнения указываются в произвольных состояниях заданий. При изменении состояния в задании осуществляется переход на соответствующий новый статус выполнения, если это необходимо.

В нижней части правой панели предусмотрены элементы управления для навигации по списку состояний, указанному в профиле заданий.
