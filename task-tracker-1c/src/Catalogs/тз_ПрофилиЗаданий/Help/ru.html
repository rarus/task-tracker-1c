Профили заданий

Для упрощения постановки заданий предусмотрены «Профили заданий» – настраиваемые шаблоны. Достаточно создать и настроить профиль один раз, чтобы использовать его в дальнейшем для множества заданий с одинаковыми параметрами.

В профиле настраиваются следующие значения:
  • Виды аналитик - настраиваемый список параметров для анализа заданий.
  • Состояния заданий – связаны со статусами выполнения заданий (Новое, В работе, Завершено).
  • Виды заданий -список доступных видов заданий в задании
  • Приоритеты - список доступных приоритетов в задании.
  • Проекты - перечень проектов, по которым ведутся задания.
